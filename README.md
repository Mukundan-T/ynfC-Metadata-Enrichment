# ynfC: Uncovering Metadata Correlations with Rare Gene Expression

I worked on this project in one of my Union College courses: CSC 483 (Applied Biological Data Science). In this project, I worked with large transcriptomic and genomic datasets, and sample metadata.

My main focus in this project was to look at experimental conditions under which ynfC (an uncharacterized lipase core gene that is rarely expressed) is likely to be highly expressed. Please refer to my Colab notebook for further details on this project!
